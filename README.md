1️⃣ What is the difference between var, let, and const?
Ans: var is function scoped and can be redeclared.
let is block scoped and can be updated but not redeclared.
const is block scoped and cannot be updated.

2️⃣ What is the spread operator (...)?
Ans: The spread operator (...) expands elements from arrays or objects.
Example: const arr = [1,2];
const newArr = [...arr,3];

3️⃣ What is the difference between map(), filter(), and forEach()?
Ans: map returns a new array.
filter returns elements that match a condition.
forEach loops through elements but returns nothing.

4️⃣ What is an arrow function?
Ans: Short syntax for writing functions.
Example:
const add = (a,b) => a+b;

5️⃣ What are template literals?
Ans: Template literals allow embedding variables inside strings.
Example:
const name = "Riad";
console.log(`Hello ${name}`);
